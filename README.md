# DraftLab Scaffold

A modern web development starter built with Astro v5, Tailwind CSS v4, and React v19. This scaffold provides a minimal foundation for building fast, content-focused websites with component-based architecture and utility-first styling.

## Quick Start

```sh
npm install
npm run dev
```

## Features

- **Astro v5** - Islands architecture with partial hydration
- **Tailwind CSS v4** - Utility-first styling with automatic class ordering
- **React v19** - Interactive component support
- **MDX** - Markdown with JSX components
- **Netlify Ready** - Preconfigured for deployment

## Commands

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
